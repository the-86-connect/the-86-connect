Security & Reliability

Enterprise-Grade Protection

Multi-layered security architecture protecting user data and system integrity.

🛡️

CSRF Protection

Double-submit cookie pattern on all mutating requests. Cross-subdomain support via .the86connect.com domain.

⏱️

6 Rate Limit Tiers

Granular limits from 3 req/2min (admin login) to 100 req/15min (general API). IP-based submission tracking with cooldowns.

🔒

Brute-Force Protection

LoginAttempt table tracks IP + email, 15-minute lockout after 3 consecutive admin failures. Rate limits skip successful logins.

🤖

Bot Protection

Honeypot field detection on all form submissions. Applied to study application, sourcing inquiry, and consultation routes.

🍪

Secure Cookies

httpOnly, Secure (production), SameSite=Lax, Domain scope (.the86connect.com) for cross-subdomain JWT sessions.

🔍

Input Validation

Zod validation on both frontend (React Hook Form) and backend (server-side re-validation). SQL injection prevented via Prisma ORM.

📊

Connection Pooling

PostgreSQL connection pool with 20 max connections and 10s timeout. Transaction timeouts prevent hung queries from blocking all requests.

🗄️

Memory Protection

In-memory tracker maps capped at 10,000 entries to prevent memory leaks on long-running servers. Automatic oldest-entry eviction.

📝

Request Logging

Structured JSON logs for every request: method, path, status, duration, IP. Slow queries (>1s) and errors flagged for debugging.

🔄

DB Retry at Startup

Database connection retry with 3 attempts and 5s delay. Prevents crash-loop on Render if PostgreSQL is temporarily unavailable.

🛑

Error Boundary

React ErrorBoundary component catches UI crashes, shows fallback with "Try Again" button. Sentry integration for client-side error tracking.

⏳

Graceful Shutdown

SIGTERM handling with 9s timeout. Closes HTTP server, disconnects Prisma, exits cleanly. Prevents data corruption during deploys.
